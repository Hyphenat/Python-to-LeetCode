## What I Learned

* Lists store multiple values in a single variable.
* Python list indexing starts from 0.
* Negative indexing can access elements from the end of a list.
* `len()` returns the number of elements in a list.
* `range(len(nums))` helps access both index and value.
* `append()` adds an element to the end of a list.
* `pop()` removes the last element.
* Loops can be used to find sums, maximum values, and count elements satisfying a condition.
